## 🔨 개발툴

-   Server: AWS EC2 (Ubuntu 18.04 LTS)
-   Database : MongoDB
-   Runtime Flatform : Node.js
-   Tool : Git, Notion

<br/>
  
  ## 🛠 구현한 기능 
<li> <b> 로그인 / 회원가입 </b><br/>
: JWT를 이용한 로그인과 회원가입<br/>

<br/>

<li> <b> 메인페이지 </b><br/>
: 사용자의 거주 지역에 해당하는 판매 글을 볼 수 있습니다. <br/>
: 판매 게시글은 기본적으로 이미지 썸네일과 제목, 가격이 표기되며, 해당 상품을 관심상품으로 등록한 모든 사용자들의 하트의 개수가 함께 디스플레이 됩니다. <br/>
